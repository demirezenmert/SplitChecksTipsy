
# SplitChecksTipsy

![License](https://img.shields.io/badge/license-MIT-blue.svg)

SplitChecksTipsy is a simple web application designed to help friends split their restaurant bills more easily and accurately, especially when it comes to handling tips and taxes. This repository contains the source code and documentation for the application.

## Features

- Calculate individual shares of a restaurant bill including taxes and tips.
- Split the bill equally among friends.
- Customize tip percentage rate.
- User-friendly and responsive web interface.

## Getting Started

### Prerequisites

To run SplitChecksTipsy locally, you need the following software installed on your system:

- Node.js: [Download and Install Node.js](https://nodejs.org/)
- Git: [Download and Install Git](https://git-scm.com/)

### Installation

1. Clone this repository to your local machine using:

   ```bash
   git clone https://github.com/demirezenmert/SplitChecksTipsy.git
   ```

2. Navigate to the project directory:

   ```bash
   cd SplitChecksTipsy
   ```

3. Install the project dependencies:

   ```bash
   npm install
   ```

## Usage

1. Start the development server:

   ```bash
   npm start
   ```

2. Open your web browser and visit `http://localhost:3000` to use the application locally.

3. Enter the bill amount, specify the number of friends, and adjust the tip  as needed.

4. Click the "Calculate" button to split the bill, and the results will be displayed on the screen.

5. Enjoy effortlessly splitting restaurant bills with your friends!
